# Privacy Policy for Google Chrome Extensions

🌍 **Bilingual Privacy Policy Documents** - English & Chinese  
👨‍💼 **Personal Developer Edition**

This repository contains comprehensive, professional privacy policy documents for Chrome browser extensions. Available in multiple formats to meet various deployment scenarios.

---

## 📦 What's Included

| File | Format | Content | Best For |
|------|--------|---------|----------|
| `PRIVACY_POLICY.md` | Markdown | Bilingual (EN + ZH) | GitHub, Documentation |
| `privacy-policy.html` | Interactive HTML | Bilingual (EN + ZH) | Websites, Web Display |
| `PRIVACY_POLICY_EN.txt` | Plain Text | English Only | Chrome Web Store |
| `PRIVACY_POLICY_ZH.txt` | Plain Text | Chinese Only | Chinese Users |
| `USAGE_GUIDE.md` | Guide | Instructions | Reference |

---

## 🚀 Quick Start

### For Chrome Web Store:
Copy content from `PRIVACY_POLICY_EN.txt` → Chrome Developer Dashboard → Privacy Policy field

### For Website:
Deploy `privacy-policy.html` → Share the URL → Users can toggle languages

### For Documentation:
Reference `PRIVACY_POLICY.md` → Add to GitHub → Link from README
