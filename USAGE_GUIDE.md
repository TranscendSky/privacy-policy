## 🚀 How to Use / 如何使用

### For Chrome Web Store Submission:
1. Use **PRIVACY_POLICY_EN.txt** for the privacy policy field if you're submitting in English
2. Copy the entire content from the file
3. Paste it into the "Privacy Policy" field in Chrome Web Store Developer Dashboard
4. Make sure all text is visible and properly formatted

**Chrome Web Store的提交流程:**
1. 使用 **PRIVACY_POLICY_EN.txt** 的内容
2. 复制全部文本
3. 粘贴到 Chrome Web Store 开发者仪表板的"隐私政策"字段
4. 确保所有文本可见且格式正确

### For Website Display:
1. Use **privacy-policy.html**
2. Deploy the file to your web server
3. Update the URL in your extension or website
4. Users can toggle between English and Chinese with buttons

**在网站上显示:**
1. 使用 **privacy-policy.html**
2. 将文件部署到您的Web服务器
3. 在您的扩展或网站中更新URL
4. 用户可以使用按钮在英文和中文之间切换

### For Documentation:
1. Use **PRIVACY_POLICY.md**
2. Add to your GitHub repository
3. Include link in your README.md
4. Works well with GitHub's markdown rendering

**用于文档:**
1. 使用 **PRIVACY_POLICY.md**
2. 添加到您的GitHub存储库
3. 在README.md中包含链接
4. 适用于GitHub的markdown渲染

### For Email or Distribution:
1. Use **PRIVACY_POLICY_EN.txt** or **PRIVACY_POLICY_ZH.txt** depending on the language
2. Copy content to email or document
3. Attach or include as link

**通过电子邮件或分发:**
1. 根据语言选择 **PRIVACY_POLICY_EN.txt** 或 **PRIVACY_POLICY_ZH.txt**
2. 复制内容到电子邮件或文档
3. 作为附件或链接包含
